Add these files for the app icon and splash:

- `icon.png` – App icon (1024x1024)
- `splash-icon.png` – Splash screen icon (recommended 1024x1024)
- `adaptive-icon.png` – Android adaptive icon foreground (1024x1024)
- `favicon.png` – Web favicon (48x48)

You can use placeholder images for development. Expo will warn if they're missing.
